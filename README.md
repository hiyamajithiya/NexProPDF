## üöÄ Installation Instructions

### Step 1: Install Python (if not already installed)

1. Download Python 3.8 or higher from: https://www.python.org/downloads/
2. Run the installer
3. **IMPORTANT:** Check the box "Add Python to PATH"
4. Click "Install Now"
5. Verify installation by opening Command Prompt and typing: `python --version`

### Step 2: Install NexPro PDF

1. Extract the NexPro PDF zip file to your desired location
2. Double-click on **`INSTALL.bat`**
3. Wait for all dependencies to install (may take 3-5 minutes)
4. When you see "Installation Complete!", you're ready to use NexPro PDF

### Step 3: Run NexPro PDF

- Double-click on **`RUN_NexPro.bat`** to start the application
- The application will open centered on your screen

## üìñ Usage Tips

### Opening PDFs
1. Click "Open" in File tab, or
2. Use File ‚Üí Open from menu bar, or
3. Press Ctrl+O

### Text Editing
1. Click "Add Text" in Edit tab
2. Click anywhere on the PDF where you want to add text
3. Type your text in the edit box
4. Press Ctrl+Enter to save, or Esc to cancel

### Page Navigation
- Use Previous/Next buttons
- Use Page input field to jump to specific page
- Use zoom controls to adjust view

### Saving Changes
- Always save your work using "Save" or "Save As"
- Original file is never modified until you save
